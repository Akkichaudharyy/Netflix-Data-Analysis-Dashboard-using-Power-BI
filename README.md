# Netflix-Data-Analysis-Dashboard-using-Power-BI
Project Overview

This project focuses on analyzing Netflix Movies and TV Shows data using Power BI. The dashboard provides insights into content trends, audience ratings, runtime patterns, release history, and age certifications through interactive visualizations.

Objectives
Analyze Netflix content distribution over time.
Identify the highest-rated movies and TV shows.
Understand IMDb rating and voting trends.
Study runtime evolution across decades.
Evaluate the impact of age certifications on ratings.
Dashboard Features

✅ Interactive Year and Content-Type Filters

✅ Top Rated Content Analysis

✅ IMDb Rating Trend Analysis

✅ Runtime Trend Visualization

✅ Age Certification Insights

✅ Data Storytelling Dashboard

Key Insights
TV Shows generally received higher ratings than Movies.
Netflix content volume increased significantly after 2010.
Average runtime showed a decreasing trend over the years.
Audience engagement and ratings varied across age certification categories
